# 投行业务参与助手

## 功能定位

本skill面向非投行部门普通员工，基于投行30个业务模型，提供以下核心能力：

### 1. 业务推荐
根据客户基本信息（行业、规模、发展阶段、资金需求等），智能匹配适合的投行业务类型。

### 2. 业务教学
用户选择具体业务后，获得该业务的：
- 适用情形判定标准
- 关键要素清单
- 客户需准备的材料
- 常见关注点
- 操作步骤说明

### 3. 支持人员匹配
各业务标注对应的投行部门支持人员，普通员工可获知：
- 业务负责人姓名/工号
- 联系方式
- 支持范围

## 业务推荐逻辑

### 输入维度
用户需提供客户以下信息（可选填）：

| 维度 | 具体要素 | 用途 |
|-----|---------|-----|
| 企业性质 | 上市公司/非上市公司/国有企业/民营企业 | 初筛业务范围 |
| 行业属性 | 所属行业、主营业务、产业链位置 | 判断政策适用性 |
| 发展阶段 | 初创/成长/成熟/转型/困境 | 匹配业务类型 |
| 资金需求 | 金额、期限、用途、紧急程度 | 选择产品类型 |
| 股权结构 | 实控人比例、股权集中度 | 判断可行性 |
| 财务概况 | 资产规模、盈利能力、负债水平 | 评估准入条件 |
| 特殊情形 | 并购意向、重整需求、合规问题 | 定向推荐 |

### 匹配规则

#### 股权类触发条件
- 上市公司 + 有融资需求 → 推荐定增、可转债
- 非上市公司 + 业绩达标 + 合规 → 推荐IPO
- 现有股东 + 资金紧张 → 推荐定增

#### 债权类触发条件
- 企业有融资需求 + 有存量债务 → 推荐公司债置换
- 国有企业 + 短期资金需求 → 推荐银行间债务工具
- 有优质资产 + 盘活需求 → 推荐ABS

#### 并购类触发条件
- 上市公司 + 扩张需求 → 推荐上市公司收购资产
- 资金充裕 + 产业整合意向 → 推荐收购上市公司
- 资不抵债 + 有挽救价值 → 推荐破产重整

## 教学模块设计

### 定增业务教学

#### 适用情形判定
**业务分析模型要点：**
1. 股东与实控人：实控人持股比例宜在30%-60%，股权质押比例不超过50%
2. 现金与负债：货币资金相对充裕，资产负债率处于行业合理区间
3. 市场地位与业绩增速：行业地位稳固，近三年扣非净利润为正或具备成长性
4. 融资用途：符合国家产业政策，投向实体经济
5. 舆情与诉讼处罚：近三年无重大违法违规，无持续经营不确定性

**方案设计模型要点：**
1. 发行规模：根据融资需求确定，不超过发行前总股本的30%
2. 发行价格：不低于定价基准日前20个交易日均价的80%
3. 发行时机：市场环境平稳、股价处于合理区间

**交叉验证模型要点：**
1. 治理结构与实控人：是否存在控制权争议
2. 产品、客户与供应商：是否存在重大依赖
3. 财务数据异常：收入成本匹配性、应收账款合理性
4. 募集资金：是否存在变更用途历史
5. 舆情与诉讼处罚：持续跟踪负面信息

#### 客户需准备的材料
- 营业执照、公司章程
- 近三年审计报告及最近一期财务报表
- 股东大会决议、董事会决议
- 募集资金可行性研究报告
- 法律意见书
- 信用评级报告（如有）

#### 常见关注点
- 发行定价与市场接受度
- 现有股东认购意愿
- 募集资金使用进度
- 项目收益实现情况

---

### IPO业务教学

#### 适用情形判定
**业务分析模型要点：**
1. 合规性：公司治理健全，内控有效，无重大违法违规
2. 盈利能力：连续三年盈利且净利润累计不低于1.5亿元（主板）
3. 成长性：营业收入持续增长，具备核心竞争优势
4. 独立性：资产完整、人员独立、财务独立、机构独立、业务独立
5. 募集资金用途：投向主营业务，符合产业政策

**方案设计模型要点：**
1. 板块选择：主板（大盘蓝筹）/科创板（硬科技）/创业板（三创四新）/北交所（专精特新）
2. 发行时机：市场窗口期、行业景气度、企业业绩释放期
3. 定价策略：参考可比公司估值，结合市场接受度

**交叉验证模型要点：**
1. 同业竞争：控股股东、实控人及关联方是否存在竞争
2. 关联交易：必要性、公允性、决策程序
3. 财务异常：收入确认政策、成本结转、费用归集
4. 法律合规：诉讼仲裁、行政处罚、知识产权

#### 客户需准备的材料
- 整套工商登记资料
- 近三年及一期审计报告
- 法律尽职调查报告
- 资产评估报告（如有）
- 募集资金运用可行性研究报告
- 招股说明书（由投行撰写，企业提供素材）

#### 常见关注点
- 上市板块选择依据
- 发行估值合理性
- 历史沿革问题清理
- 关联交易规范

### 公司债业务教学

#### 适用情形判定
**业务分析模型要点：**
1. 主体评级：一般需AA及以上评级
2. 偿债保障：经营性现金流稳定，可覆盖利息支出
3. 募集资金用途：合规用途（补充流动资金、偿还债务、项目建设等）
4. 行业属性：符合国家产业政策，非限制类行业

**方案设计模型要点：**
1. 发行规模：净资产40%以内（扣除已发行债券余额）
2. 期限结构：1-10年，匹配资金用途
3. 票面利率：参考同期可比债券，结合市场供需

**交叉验证模型要点：**
1. 财务指标：资产负债率、流动比率、利息保障倍数
2. 担保增信：第三方担保、资产抵押、差额补足
3. 偿债来源：经营性现金流、货币资金、再融资能力

#### 客户需准备的材料
- 营业执照、公司章程
- 近三年审计报告
- 信用评级报告
- 募集资金用途说明
- 董事会/股东会决议

#### 常见关注点
- 发行窗口选择
- 利率走势判断
- 投资者结构
- 后续信息披露

---

### ABS业务教学

#### 适用情形判定
**业务分析模型要点：**
1. 基础资产质量：权属清晰、可特定化、能产生稳定现金流
2. 现金流稳定性：历史现金流表现良好，未来可预测
3. 原始权益人资质：持续经营能力，无重大信用风险

**方案设计模型要点：**
1. 交易结构：优先级/次级分层、循环购买（如有）
2. 分层设计：优先档评级要求、次级档厚度
3. 信用增级：内部增级（超额抵押、准备金账户）+ 外部增级（担保、差额支付）

**交叉验证模型要点：**
1. 现金流测算：压力测试、早偿/逾期影响
2. 底层资产：穿透核查、权属确认
3. 持续督导：资产服务机构履职、信息披露

#### 适用基础资产类型
- 应收账款（贸易应收、工程应收）
- 租赁债权（融资租赁、经营租赁）
- 信贷资产（个人消费贷、小微企业贷）
- 基础设施收费权（高速公路、供水供热）

#### 常见关注点
- 基础资产集中度
- 循环购买机制（如有）
- 加速清偿触发条件
- 资产服务机构更换

### 上市公司收购资产业务教学

#### 适用情形判定
**业务分析模型要点：**
1. 标的质量：盈利能力、成长性、行业地位
2. 协同效应：产业协同、渠道协同、技术协同
3. 估值合理性：市盈率、市净率与可比交易对比
4. 支付方式：现金/股份/混合，考虑股权稀释

**方案设计模型要点：**
1. 对价安排：定价依据、业绩承诺、补偿机制
2. 业绩承诺：承诺期、承诺利润、补偿方式
3. 整合方案：治理整合、业务整合、人员整合

**交叉验证模型要点：**
1. 关联交易：是否构成、审议程序、回避表决
2. 同业竞争：收购后是否产生、解决措施
3. 信息披露：敏感期、内幕信息知情人登记

#### 客户需准备的材料
- 标的资产评估报告
- 审计报告（备考报表）
- 法律尽职调查报告
- 收购报告书/重组报告书
- 独立董事意见

#### 常见关注点
- 标的估值溢价合理性
- 业绩承诺可实现性
- 商誉减值风险
- 整合效果

---

### 收购上市公司业务教学

#### 适用情形判定
**业务分析模型要点：**
1. 标的公司估值：市盈率、市净率、市值规模
2. 收购资金来源：自有资金、并购贷款、定增配套
3. 控制权安排：收购比例、一致行动协议、表决权委托

**方案设计模型要点：**
1. 收购方式：协议转让、要约收购、二级市场增持、间接收购
2. 定价策略：溢价幅度、分期支付安排
3. 融资结构：并购贷款比例、期限、担保

**交叉验证模型要点：**
1. 一致行动：收购方内部及关联方一致性
2. 要约义务：触发要约收购时的应对
3. 合规性：信息披露、敏感期交易、短线交易

#### 客户需准备的材料
- 收购方资金来源证明
- 标的公司尽职调查资料
- 收购报告书/要约收购报告书
- 财务顾问核查意见

#### 常见关注点
- 收购溢价与协同效应匹配
- 融资结构与偿债压力
- 标的公司治理整合
- 中小股东权益保护

### 破产重整业务教学

#### 适用情形判定
**业务分析模型要点：**
1. 重整可行性：持续经营价值、行业前景、挽救必要性
2. 出资人权益调整：股权让渡比例、资本公积转增
3. 债权清偿方案：债权分类、清偿比例、清偿方式

**方案设计模型要点：**
1. 重整计划：经营方案、投资人引入、债权清偿安排
2. 经营方案：主营业务恢复、资产处置、债务减免
3. 投资人引入：产业投资人、财务投资人、投资条件

**交叉验证模型要点：**
1. 债权债务核查：申报债权审查、职工债权调查
2. 资产评估：市场法、收益法、成本法综合评估
3. 表决通过：债权人分组表决、出资人组表决、法院批准

#### 适用情形
- 上市公司重整（保壳需求）
- 大型民营企业债务重组
- 具有产业价值的困境企业

#### 常见关注点
- 重整价值与清算价值比较
- 债权人利益平衡
- 重整计划执行可行性
- 退市风险警示

### 买方分析业务教学

#### 功能定位
帮助识别和筛选潜在投资者，分析其投资需求与认购能力。

#### 分析维度
1. 投资风格：价值投资/成长投资/指数跟踪
2. 资金规模：管理资产规模、可用资金量
3. 投资限制：行业偏好、单只股票比例限制
4. 历史偏好：过往参与定增、可转债投资记录

#### 应用场景
- 定增发行前投资者沟通
- 可转债路演对象筛选
- 战略配售投资者锁定

---

### 信披判断业务教学

#### 功能定位
辅助判断信息披露义务的触发时点、披露内容及合规要求。

#### 判断维度
1. 披露时点：事件发生时点、董事会审议时点、协议签署时点
2. 披露内容：事实要素、财务影响、风险提示
3. 披露方式：临时公告、定期报告、专项说明
4. 合规性：是否属于重大事件、是否达到披露标准

#### 适用情形
- 重大合同签署
- 对外担保
- 关联交易
- 重大诉讼仲裁

---

### 临时公告生成业务教学

#### 功能定位
根据披露事项的事实依据，辅助生成符合格式要求的临时公告文本。

#### 生成要素
1. 公告类型：重大事项/关联交易/对外担保/其他
2. 事实依据：合同文本、董事会决议、评估报告
3. 披露要点：时间、金额、影响、风险提示

#### 输出内容
- 公告标题
- 公告正文（含必备要素）
- 附件清单

### 场景2：业务教学

**用户输入：** 定增业务怎么做？

**Skill响应：**
**定增业务操作指引**

**一、业务概述**
定向增发是指上市公司向特定投资者非公开发行股票募集资金的行为。根据2023年注册管理办法，发行价格不低于定价基准日前20个交易日均价的80%，发行对象不超过35名。

**二、适用情形判定**
请协助客户确认以下要素：
1. 实控人持股比例宜在30%-60%之间，股权质押比例不超过50%
2. 公司近三年扣非净利润为正，或虽为负但具备成长性
3. 募集资金用途符合国家产业政策，投向实体经济
4. 近三年无重大违法违规记录

**三、关键操作步骤**
1. **方案论证阶段**：与公司管理层沟通融资规模、用途、定价策略
2. **尽职调查阶段**：收集财务、法律、业务资料，出具尽调报告
3. **申报准备阶段**：准备申报材料，通过内核，提交交易所审核
4. **发行阶段**：确定发行对象，签署认购协议，资金到账验资
5. **持续督导阶段**：募集资金使用跟踪，定期披露进展

**四、客户需准备的材料**
- 营业执照、公司章程
- 近三年审计报告及最近一期财务报表
- 股东大会决议、董事会决议
- 募集资金可行性研究报告
- 法律意见书、信用评级报告（如有）

**五、投行支持**
张明（工号8012）负责该业务，可提供方案设计、发行定价等专业支持。
